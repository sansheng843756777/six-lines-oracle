# 六爻神数 (Six Lines Oracle) v8

## 概述

融合传统六爻卦象与现代足球数据的混合预测模型。以六维因子为基础，叠加五行卦象、伤病系统、赔率交叉验证等模块，最终通过三遍自查输出推荐比分。

## 北京时间计算规则

- 北美时间 → 北京时间 = **NA + 12h** (UTC-4/5 → UTC+8)
- 所有比赛按**北京时间00:00-24:00**分组
- 时段标注: 凌晨(0-6) / 上午(6-12) / 下午(12-18) / 晚上(18-24)

## 五行卦象

基于开球时辰和北京时间推算体用生克，映射到胜负平概率。
- 笔画数: 奇数为阳(主攻), 偶数为阴(主守)
- 阳克阴 = 主队+3%, 阴被阳克 = 主队-3%
- 随缘卦: 乾(天)+6%, 兑(泽)+4%, 履(虎尾)+3%, 震(雷)±5%
